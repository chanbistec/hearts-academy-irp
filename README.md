## Program Overview

Hearts Academy's Industry Readiness Program takes undergraduates from classroom theory to deployed, production-grade software — in 24 weeks.

- **Duration:** 6 months (24 weeks, 5 phases)
- **Commitment:** 15–20 hours/week
- **Cohort Size:** 8–10 students
- **Technologies:** React, TypeScript, Node.js, PostgreSQL, AI Agents, Context Engineering
- **Application:** [Apply Now](https://forms.office.com/r/kM2prJrNgg)
- **Website:** [hearts.academy](https://www.hearts.academy)

## Key Value Props

1. **Real projects, not exercises** — Ship production software
2. **AI-native curriculum** — Agentic era, context engineering, multi-agent orchestration
3. **Senior mentorship** — Code reviews and career guidance from experienced engineers
4. **Performance-based pricing** — Top 10% get 100% fee waiver, top 25% get 50%
5. **Company placement** — Real work environment, real expectations
6. **Portfolio over transcript** — Graduate with GitHub proof of competence
